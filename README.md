# Fehlerkorrektur in digitalen Nachrichten am Beispiel von Reed-Solomon-Codes

## Wie man das Beispielprogramm ausführt
Folgen Sie [dieser Anleitung](https://wiki.python.org/moin/BeginnersGuide/Download) für ihr jeweiliges Betriebssystem.\
Installieren Sie ggf. SymPy mit ```pip install sympy```.\
Öffnen Sie ein Terminal im Ordner, in dem sich auch die Programmdateien befinden.\
Führen Sie den Befehl ```python <Dateiname>.py``` aus.
Sollten Sie zwei verschiedene Python-Versionen installiert haben, müssen Sie wahrscheinlich ```python3``` anstatt von ```python``` verwenden.
